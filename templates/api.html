Common API Endpoints

Review List

GET /api/reviews

Get a list of reviews, supports pagination and rating filters

Review Details

GET /api/reviews/{review_id}

Get detailed information for a specific review

Product Reviews

GET /api/product/{product_id}

Get all reviews for a specific product

User Reviews

GET /api/user/{user_id}

Get all reviews by a specific user

Natural Language Query

POST /api/query

Query the review dataset using natural language

Usage Examples

Use curl to query reviews with a 5-star rating:

curl "http://localhost:5000/api/reviews?min_score=5&limit=10"

Use natural language query:

curl -X POST "http://localhost:5000/api/query" \
  -H "Content-Type: application/json" \
  -d '{"query": "Find 5-star reviews for chocolate"}'
