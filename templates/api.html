常用 API 端點

評論列表

GET /api/reviews

獲取評論列表，支持分頁和評分過濾

評論詳情

GET /api/reviews/{review_id}

獲取特定評論的詳細信息

產品評論

GET /api/product/{product_id}

獲取特定產品的所有評論

用戶評論

GET /api/user/{user_id}

獲取特定用戶的所有評論

自然語言查詢

POST /api/query

使用自然語言查詢評論數據集

使用範例

使用 curl 查詢評分為5星的評論：

curl "http://localhost:5000/api/reviews?min_score=5&limit=10"

使用自然語言查詢：

curl -X POST "http://localhost:5000/api/query" \
  -H "Content-Type: application/json" \
  -d '{"query": "找出有關巧克力的五星評論"}'
